# MotorDC-acionamento-por-bot-o-com-Arduino-Uno

Neste circuito, a fonte de 9V alimenta diretamente o motor, que com a ajuda do transistor recebe o sinal do Arduino, lembrando que a sainda de um pino digital do Arduino Uno R3 é recomendado utilizar apenas 20mA. Mas neste caso, o motor que utilizei precisa aproximadamente 1A. É ai que o transistor entra, ulitilei um modelo MOSFET chamado IRF520, era o que eu tinha disponível no momento, ele ficou responsável de receber o sinal do Arduino e mandar para o motor controlando sua corrente, evitando de queimar a saída digital. E por final, acrescentei um diodo, modelo 1N4007, entre os dois polos do motor, assim evitando que a corrente fornecida pela fonte retorne e queime o circuito. A ligação do botão foi feita diretamente no Arduino, ele é apenas um sinal para fazer o motor funcionar.
